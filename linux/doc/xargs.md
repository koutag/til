# xargs

## 概要

**xargs**は標準入力に来たファイルを、コマンドライン引き数に変換して引数のプログラムに渡す
パイプは標準出力結果を次のコマンドに渡す。

```bash
ls | wc                          # 標準入力を渡せばいいだけなのでパイプでおk
find . -name "*.md" | echo       # echoは引き数を出力するのでエラー
find . -name "*.md" | xargs echo # これなら引数として扱われるので正常終了する
```
