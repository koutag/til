# 変数
```vim
let {変数} = {式}
```
* let で変数を定義出来る。
* vimは動的型付け
* 変数名には、アルファベット、数字、アンダースコアが使える。
* vimは接頭語を付けるとスコープを変えれるっぽい
    * s:name スクリプトローカル
    * b:name バッファローカル
    * w:name ウィンドウローカル
    * g:name グローバル変数
    * v:name vimが定義する変数
```vim
:unlet s:count
:unlet! s:count
```
* 変数の削除には`:unlet`コマンドを使う。
* 変数が存在しない場合でもエラーをおこさないようにするには！を付ける。
* スクリプトの実行が終了したとき、ローカル変数は自動的には削除されない。

* その他にも変数に以下の内容を設定出来る。
    * $NAME		環境変数
    * &name		オプション
    * @r		レジスタ
