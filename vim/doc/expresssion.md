# 式
* 四則演算は以下の通り(割と普通)
    * a + b		加算
    * a - b		減算
    * a * b		乗算
    * a / b		除算
    * a % b		剰余演算(余りを得る)
* 3項演算子も使える。 a ? b : c

## if文
```vim
if {condition}
    {statements}
elseif {condition}
    {statements}
else
endif
```
* vimのif文。他の言語と大体同じ。

## 論理演算子
* 演算子一覧
    * a == b		等しい
    * a != b		等しくない
    * a >  b		より大きい
    * a >= b		より大きいか等しい
    * a <  b		より小さい
    * a <= b		より小さいか等しい
    * a =~ b        パターンにマッチする(文字列限定)
    * a !~ b        パターンにマッチしない(文字列限定)

* 文字列と数値を比較するとき文字列を数値に変換する。
* 文字列が数字でない場合は0([false](boolean.md))になる。
* パターンマッチングはを使ったほうが記述しやすい[シングルクォート](string.md)
